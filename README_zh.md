# ⚡ 闪电少年英语 ⚡

五年级上册英语词汇学习应用 — 闪电少年主题

一个有趣的五年级英语词汇学习应用，采用闪电少年超级英雄主题！

## 功能特点

- 🃏 **闪卡模式** — 点击翻转，自动播放语音
- 🎯 **测验模式** — 多选题测试
- 🧩 **配对游戏** — 英中单词配对
- 📖 **单词列表** — 按单元浏览，记录掌握程度
- 🔊 **语音发音** — Edge TTS 语音合成（Andrew + 云希）
- 🌐 **双语界面** — 中英文切换
- 💾 **进度保存** — localStorage 本地存储

## 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需安装！

## 部署到 GitHub Pages

1. Fork 本仓库
2. 进入 Settings → Pages
3. 选择 `main` 分支，点击 Save
4. 访问 `https://yourusername.github.io/wunianji-english/`

## 技术栈

- HTML/CSS/JavaScript（无框架）
- Edge TTS 语音合成
- localStorage 进度保存
